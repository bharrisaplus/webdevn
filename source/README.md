# Webdevn

## Overview

```
source/
├── webdevn
│   ├── assets/ <-- non-code things used by the app
│   │   ├── doc
│   │   └── img
│   │  
│   ├── cli.nim <-- Reading/validating user input
│   ├── localserver.nim <-- Listening and handling network responses
│   ├── meta.nim <-- Values read from define flags or files in `assets/`
│   ├── scribe.nim <-- Logging logging logging
│   ├── type_defs.nim <-- Compiler/Developer hints
│   └── utils.nim <-- Other helpful uncategorized things
│  
└── webdevn.nim <-- Main module where loopin begins
```
